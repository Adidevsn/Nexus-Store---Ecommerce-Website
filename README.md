# Nexus-Store---Ecommerce-Website
⬡ NEXUS STORE
A fully functional dark-themed ecommerce web app built with Django 4.2 as a 3rd year CSE project. Features a futuristic UI with glassmorphism cards, neon accents, and smooth animations — all without a single frontend framework.

Features
Shopping

Product catalog with search, category filtering, and sort by price/date
Product detail pages with stock indicators and related products
Session-based cart — works without login, persists for 7 days
Stripe Checkout for secure payments (no card data touches your server)
Automatic stock decrement after successful orders

Accounts & Orders

User registration, login, profile with avatar and address
Order history with status timeline (Pending → Confirmed → Shipped → Delivered)
Full order detail with item breakdown
